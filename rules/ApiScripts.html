    Roll20 API Scripts

    Waterdeep: Dragon Heist Campaign

    In our current campaign I have the following API scripts installed:

    • 5th Edition OGL by Roll20 Companion:
      • Enhances the Official 5th Edition OGL by Roll20 Character Sheet. The Companion currently supports Ammo Tracking, Automatic NPC Tokens, Automatic Death Save Tracking, and Automatic Spell Slot Tracking.
      • https://wiki.roll20.net/5th_Edition_OGL_by_Roll20#Utilizing_the_5th_Edition_OGL_Companion_API_Script
    • Bump
      • Bump provides a way to invisibly manipulate tokens on the GM Layer from the Objects Layer, and vice versa.
      • https://app.roll20.net/forum/post/2231120/script-bump-v0-dot-2-4-invisible-manipulation-of-gm-layer-objects-from-the-token-layer-new-thread
    • CombatTracker:
      • CombatTracker will be a great help in battles. Easily keep tracks of who’s turn it is, use a turn timer, let players progress to the next turn by themselves, adding/removing conditions, etc.
      • https://github.com/RobinKuiper/Roll20APIScripts/tree/master/CombatTracker
    • Concentration
      • Concentration keeps track of characters concentration, and reminds them to do a concentration check.
      • https://wiki.roll20.net/Script:Concentration
    • GroupCheck:
      • This is an API script meant to run checks for several tokens at once. You can specify the type of check to run and it will roll it once for every selected token.
      • https://app.roll20.net/campaigns/scripts/3745638
    • GroupInitiative:
      • Rolls initiative for the selected tokens and adds them to the turn order if they don’t have a turn yet.
      • https://wiki.roll20.net/Script:Group_Initiative
    • StatusInfo:
      • StatusInfo works nicely together with Tokenmod and DeathTracker and InspirationTracker scripts. It shows condition descriptions whenever a statusmarker is set or when the command !condition is used, eg: !condition prone.
      • https://github.com/RobinKuiper/Roll20APIScripts/tree/master/StatusInfo
    • TokenMod
      • TokenMod provides an interface to setting almost all settable properties of a token.
      • https://app.roll20.net/forum/post/1257490/script-tokenmod-an-interface-to-adjusting-properties-of-a-token-from-a-macro-or-the-chat-area/?pagenum=1
    • Torch
      • A simple script for giving lights to tokens and turning off and on dynamic lighting.
      • https://app.roll20.net/forum/post/1104897/script-torch-a-simple-script-for-giving-lights-to-tokens-and-turning-off-and-on-dynamic-lighting
    • weatherInfo
      • Home grown adaptation of Robin Kuiper’s StatusInfo script to roll for random weather effects
